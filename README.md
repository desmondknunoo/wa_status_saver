# WhatsApp Status Saver (Android Only)

A professional, production-ready Flutter application to view and save WhatsApp statuses (Images and Videos) from both WhatsApp and WhatsApp Business.

## ğŸ— Getting Started

### Prerequisites

- Flutter SDK (latest stable version)
- Android Studio or VS Code
- An Android device or emulator

### Installation

1. Fork the repository by clicking the "Fork" button on [GitHub](https://github.com/desmondknunoo/wa_status_saver)

2. Clone your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/wa_status_saver.git
   cd wa_status_saver
   ```

3. Add upstream remote:
   ```bash
   git remote add upstream https://github.com/desmondknunoo/wa_status_saver.git
   ```

4. Install dependencies:
   ```bash
   flutter pub get
   ```

### Keeping Your Fork Updated

```bash
git fetch upstream
git checkout main
git merge upstream/main
```

### Submitting a Pull Request

1. Create a feature branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```

2. Make your changes and commit:
   ```bash
   git add .
   git commit -m "Description of changes"
   ```

3. Push to your fork:
   ```bash
   git push origin feature/your-feature-name
   ```

4. Open a Pull Request on GitHub

### Running the App

```bash
flutter run
```

### Building for Release

To generate a release APK:

```bash
flutter build apk --release
```

The APK will be located at `build/app/outputs/flutter-apk/app-release.apk`.
