# Contributing to WhatsApp Status Saver

Thank you for your interest in contributing! This guide will help you get started.

## How to Contribute

### 1. Fork the Repository

Fork the repository on GitHub by clicking the "Fork" button at the top right of the [repository page](https://github.com/desmondknunoo/wa_status_saver).

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/wa_status_saver.git
cd wa_status_saver
```

### 3. Add Upstream Remote

```bash
git remote add upstream https://github.com/desmondknunoo/wa_status_saver.git
```

### 4. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### 5. Make Your Changes

- Follow the existing code style
- Write clean, well-documented code
- Add tests if applicable

### 6. Commit Your Changes

```bash
git add .
git commit -m "Description of your changes"
```

### 7. Push to Your Fork

```bash
git push origin feature/your-feature-name
```

### 8. Create a Pull Request

1. Go to the [original repository](https://github.com/desmondknunoo/wa_status_saver)
2. Click "New Pull Request"
3. Select your fork and branch
4. Fill in the PR template
5. Submit

## Pull Request Guidelines

- Keep PRs focused and small
- Describe your changes clearly
- Link any related issues
- Ensure tests pass
- Follow code style conventions

## Development Setup

1. Install Flutter SDK
2. Run `flutter pub get`
3. Run `flutter analyze` to check for issues
4. Make your changes
5. Test thoroughly before submitting

## Questions?

Open an issue for discussion or reach out to the maintainers.
